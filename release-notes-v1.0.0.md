# v1.0.0 发布说明

首次发布 Krea2 Prompt Artisan。

## 下载

下载压缩包：

```text
krea2-prompt-artisan-v1.0.0.zip
```

## 安装

解压后，把里面的 `krea2-prompt-artisan` 文件夹放到 Codex 的 skills 目录：

```text
%USERPROFILE%\.codex\skills\
```

确认存在：

```text
%USERPROFILE%\.codex\skills\krea2-prompt-artisan\SKILL.md
```

然后重启 Codex 或重新打开一个会话。

## 触发示例

```text
生成K2提示词：一个美女撑着透明雨伞在雨中侧身看旁边
```

```text
生成附图的K2提示词
```

```text
把下面的提示词转成K2提示词：雨夜街头，一个穿黑色风衣的女人站在霓虹灯下
```

## 说明

- 本技能适用于 Codex，不是 ComfyUI 节点或插件。
- 技能会生成 Krea 2 英文提示词、中文译文和本地 HTML 页面。
- HTML 页面带复制按钮，方便直接复制到 Krea 2 / Krea.ai 使用。
